# Changelog

All notable changes to Privacy Panel are listed here.

---

## 2026-05-28

### Data
- Added `parent_company` field to all 101 company entries — brands are now properly attributed to their parent organizations (e.g. Instagram → Meta, YouTube → Alphabet).
- Cleaned up company display names by separating product name from parent entity.

---

## 2026-05

### Site
- **Rebranded** from *Privacy Facts* to *Privacy Panel* — new name, new domain, updated labels and API.
- Published [methodology and rubric](/rubric) for the A–F scoring system.

---

## 2026 — Launch

- Initial release covering 101 companies.
- Privacy Panel label for each company: standardized snapshot of data collection, sharing, consumer rights, and security practices.
- A–F grades based on the published rubric.
- Embeddable SVG and HTML labels for any company.
- Public API (`/api/v1/`) for programmatic access.
