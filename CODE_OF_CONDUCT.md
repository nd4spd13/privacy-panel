# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

By participating in this project — opening issues, submitting pull requests, commenting, or otherwise engaging with the community — you agree to uphold this code.

## In short

Be respectful. Assume good faith. Critique ideas, not people. Make space for newer contributors. Disagreement is fine; making the project unpleasant for others is not.

## Reporting

If you experience or witness behavior that violates the Contributor Covenant, email **conduct@privacypanel.org**. Reports are read by the project maintainers and handled in confidence.

The full text and enforcement guidelines are at the link above.
